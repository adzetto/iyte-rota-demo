# İYTE Yolculuk · Mobil UI Konsepti

Modern, sıcak ve kontrastlı bir kampüs paylaşımlı yolculuk arayüzü. Statik HTML/CSS/JS olarak hazır; GitHub Pages ile yayınlanabilir.

## Geliştirme

- Bağımlılık yok; statik dosyalar: `index.html`, `app.js`.
- Yerelde ön izleme:
  ```bash
  npx serve .
  ```

## Yayınlama (GitHub Pages)

1. Bu repo `main` branch'ine commit/push.
2. GitHub → Settings → Pages → Source: **Deploy from a branch**, branch: `main`, folder: `/ (root)`.
3. Kayıt sonrası Pages URL'si ile site otomatik servis edilir.
